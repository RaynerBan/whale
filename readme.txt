游戏规则：
1.按“start game”（或者space键、enter键）初始化游戏，再次按开始游戏；
1.控制鲸鱼，有体力值跟空气，按紧鼠标左键/键盘Up键往上游；
2.体力靠吃鱼类补充，碰撞到章鱼、乌贼则损伤体力；
3.空气靠浮动到水面补充；
4.体力或者空气一方为0则游戏结束；
5.吃金币累计分数，游戏结束显示距离、分数；
6.空气不足、体力不足时，鲸鱼眼睛会有变化，进度条也有颜色变化，以提醒玩家。
